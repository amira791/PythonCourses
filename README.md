## Overview

This repository is designed to provide a structured approach to learning Python, from basic concepts to more advanced topics like Object-Oriented Programming (OOP) and error handling. The files are organized based on different topics or lessons, making it easy to follow the progression of the learning material.

## Courses

Here are the available courses and topics:

| File Name                          | Description                                        |
| ----------------------------------- | -------------------------------------------------- |
| `VariablesDataTypesBasicOperators.py` | Introduction to variables, data types, and operators in Python. |
| `LoopsFunctionsConditionals.py`     | Understanding loops, functions, and conditional statements. |
| `ListsTuplesDictionaries.py`        | Working with lists, tuples, and dictionaries.      |
| `FileHandling.py`                   | Handling files in Python (read/write operations).  |
| `ErrorHandling.py`                  | Exception handling in Python (try-except).         |
| `FinalProjectWithoutOOP.py`         | Final project using procedural programming techniques. |
| `FinalProjectWithOOP.py`            | Final project applying Object-Oriented Programming principles. |
| `OPP.py`                            | Basics of Object-Oriented Programming in Python.   |
| `ErrorHandling.py`                  | A course focused on handling errors and exceptions in Python. |

Each file contains tasks and examples that help build your understanding of these topics.

## How to Use

1. **Clone the repository**:
    ```bash
    git clone https://github.com/amira791/Python-Courses.git
    cd Python-Courses
    ```

2. **Navigate through the course files** to learn specific topics.

3. **Run the Python files** to practice and solve the tasks provided within each course.

    Example:
    ```bash
    python3 VariablesDataTypesBasicOperators.py
    ```
